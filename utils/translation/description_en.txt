Summarize and translate web pages with one click using Google Gemini or any OpenAI-compatible LLM (OpenAI, OpenRouter, OpenCode, Ollama, self-hosted endpoints).

Features:
- One-click page summarization
- Select text and translate in one click
- YouTube caption summarization
- Follow-up questions with full conversation context
- Streaming output
- Shortcut key Alt+G (customizable)
- Custom actions: Alt+1 / Alt+2 / Alt+3

Bring your own API key. Data is sent only to the API endpoint you configure.

Zero trackers. Zero ads. Built for researchers and engineers.
